Welcome to the template v1.0

This is a template for starting web projects. The basic folder structure has already been created, as well as a standardized html basic framework. Most of the advantages come from the already added css library . It uses scss, reset css, the grid system of bootstrap, spacer classes and a solid basic structure for scss projects.

So what are you waiting for? Start faster, copy and modify.
